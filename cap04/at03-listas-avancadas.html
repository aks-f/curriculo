Ranking de tecnologias

  1. CSS

    (Folhas de Estilo em Cascata), é uma linguagem de estilo utilizada no desenvolvimento web para definir a apresentação visual de documentos estruturados em HTML.

  2. Git e GitHub

    Ferramenta fundamental para controle de versão e colaboração em projetos de software.

    Armazenar informações sobre um domínio específico (clientes, produtos, transações) para que o sistema possa recuperá-las depois.

    A linguagem essencial para interatividade no navegador e desenvolvimento fullstack.

  5. HTML

    A base de qualquer página web, definindo a estrutura e semântica do conteúdo.

Glossário de termos técnicos

HTML
HyperText Markup Language; é a linguagem usada para estruturar e organizar o conteúdo de páginas na internet.
HTTP
Hypertext Transfer Protocol; é o protocolo de comunicação fundamental da internet.
DOM
Document Object Model; é uma interface que o navegador utiliza para representar o seu código HTML como uma árvore de objetos hierárquica.
URL
Uniform Resource Locator; trata-se do endereço digital único utilizado para localizar e acessar páginas, sites, imagens ou arquivos na internet.
Servidor
Hardware: Uma máquina física (um computador potente) conectada à internet que armazena os arquivos de um site e os entrega aos usuários.
Software: O programa que roda nessa máquina (como o Apache ou Nginx) responsável por processar as requisições que chegam do navegador.
Router (Roteador)
Infraestrutura: O equipamento físico ou virtual que direciona os pacotes de dados da internet até o servidor correto.
Front-end/Back-end: A biblioteca de código responsável por ler a URL do navegador e carregar a página correspondente no site.
Framework
Biblioteca
Ambos são códigos pré-escritos para facilitar a criação de sites.
Internet
Rede
Ambos fazem referência a infraestrutura global de computadores interconectados que usam protocolos padronizados para comunicação.
